# 毛选行动力系统 (Maoxuan Action Force)

> "多少事，从来急；天地转，光阴迫。一万年太久，只争朝夕。" -- 毛泽东

基于毛选六大核心思想的个人行动力系统。将毛泽东的战略思想转化为日常工作学习的行动框架和自我诊断工具。

## 六大核心思想

| 思想 | 核心应用 | 解决什么 |
|------|---------|---------|
| **矛盾论** | 找到主要矛盾 | 任务太多，不知从何下手 |
| **集中优势兵力** | 拆解到极小，各个歼灭 | 任务太大，不敢开始 |
| **实践论** | 先做烂版，螺旋上升 | 完美主义，迟迟不动 |
| **论持久战** | 三阶段：防御→相持→反攻 | 间歇性鸡血，坚持不住 |
| **运动战** | 打得赢打，打不赢走 | 遇卡壳就全盘崩溃 |
| **纠正错误思想** | 理性复盘代替情绪内耗 | 深度自责但不改变 |

## 快速开始

### 1. 安装为 Hermes Skill

```bash
# 复制到 Hermes skills 目录
cp -r . ~/.hermes/skills/productivity/maoxuan-action-force
```

### 2. 生成每日计划

```bash
# 生成今日作战计划
python3 scripts/daily_action.py --plan

# 生成今日复盘
python3 scripts/daily_action.py --review
```

### 3. 作为 ClawHub Skill

通过 ClawHub 搜索 `maoxuan-action-force` 直接安装。

## 行动力启动协议

当你感到极度拖延时，按以下步骤执行：

1. **停下来**。拿出白纸
2. **识别主要矛盾**：今天哪件事不做会让你内心惶恐？
3. **拆解到极小**：切成5分钟以内的微型任务
4. **立刻动手**：不做任何准备
5. **初战必胜**：完成后确认"原来没那么难"
6. **继续吃掉下一块**：打一仗，吃一口
7. **卡壳就转移**：运动战寻找突破口
8. **日终复盘**：冰冷客观，不带情绪

## 适用场景

- 日常任务规划、大型项目管理
- 克服拖延、完美主义卡壳
- 遭遇瓶颈、长期坚持失败
- 每日复盘反思、归因外部环境
